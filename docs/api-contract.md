# API Sözleşmesi

Bu proje için herhangi bir backend entegrasyonu bulunmamaktadır. Proje, tamamen statik frontend teknolojileri (HTML, TailwindCSS, JavaScript) kullanılarak geliştirilmiş bir landing page'dir. Bu nedenle bir backend API endpoint listesi, request/response örnekleri mevcut değildir ve API sözleşmesi gereksizdir.

Eğer ileride bir backend entegrasyonu planlanırsa, bu doküman güncellenecektir.
